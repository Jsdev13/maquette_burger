
# 🍔 Maquette Burger

Une intégration web simple et responsive d'un site de restaurant de burgers.

## 🚀 Fonctionnalités
- Site entièrement responsive (adapté aux mobiles et tablettes).
- Menu de navigation interactif pour les mobiles.

## 🛠️ Technologies utilisées
- **HTML5** : Pour la structure de la page.
- **CSS3** : Pour le design (utilisation de Flexbox et CSS Grid).

## 📦 Comment lancer le projet ?
1. Clonez le dépôt :
   ```bash
   git clone [https://github.com/Jsdev13/maquette_burger.git](https://github.com/Jsdev13/maquette_burger.git)

## 📦 Accédez au dossier du projet ?
2. Taper sur le terminal : 
   ```bash
   cd maquette_burger
   ```
## 📦 Lancez le serveur local en PHP ?
3. Ensuite :
   ```bash
   php -S localhost:8000
   ```
